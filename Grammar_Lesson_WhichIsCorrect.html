Why this matters 🌍

Last unit your grammar jumped to 6 out of 7 — the standard conditionals and "read the result half first" really worked. 🎉 This lesson keeps that exact method and adds the one question type that's still tricky: "Which sentence is CORRECT?" Instead of hunting for the mistake, you hunt for the clean one — same skills, flipped around.

Same winning method, one new move ✅

You already know how to spot an error. "Which is correct?" just means: check each sentence, cross out the broken ones, and the survivor is your answer. We'll make that a quick, sure routine. Next →

Quick recap — the two you'll see most 📋

TypeRecipeExample
Second
(dream now)		If + past simple, … would + verbIf I had a dog, I would walk it.
Third
(past regret)		If + had + past participle, … would have + verbIf I had trained, I would have won.
Read the RESULT half first:
plain "would …" = Second · "would have …" = Third — then the if-half must match.

The three ways a sentence goes WRONG 🚫

1
"will" in the result. A Second or Third never uses will. "…it will stay dry" → wrong.
2
A broken result verb. "would" must be followed by a plain verb (Second) or "have + verb" (Third). "they would caught" / "he would noticed" → wrong.
3
Halves don't match. "had a pool" (Second) with "would have swum" (Third) → wrong. Same type both sides, or it's an error.

A correct sentence passes all three: no "will", a proper verb, and matching halves.

Drill 1 — correct or error? 🔎

Run the three checks on each one.

1. "If the shed had a lock, the tools would have been safe."
2. "If the bus had arrived, we would have caught the train."
3. "If the garden had a fence, the dog would stay in."
4. "If the cake had been iced, it will look prettier."
5. "If the team had practised, they would won the final."
6. "If the house had a garage, we would park inside."
