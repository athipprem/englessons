Section 1

💡 What Is a Conditional Sentence?

A conditional sentence has two parts: a condition (the if part) and a result (what happens because of that condition).

Think of it like a machine: you press a button (the condition) and a light turns on (the result). The word if starts the condition.

Structure

Every conditional = IF clause (condition) + Result clause (what happens). The IF clause can go first or second in the sentence — the comma moves but the meaning stays the same.

Example: "If you press the button, the light turns on." = same as "The light turns on if you press the button."

The Most Common Mistake — Unit 1

Both Zero and First Conditionals use present simple in the IF clause. The difference is the result clause: Zero uses present simple (it boils), First uses will + base (it will boil). That one word — will — is the tell.

Section 3

🔍 How to Identify Which One It Is

Step 1: Look at the result clause

The IF clause is the same for both (present simple). So ignore the IF clause first and look at the result clause.

  • Result clause has will/won't → First Conditional
  • Result clause has present simple (no will) → Zero Conditional

Step 2: Ask the meaning question

Is this sentence about something that is always true (like a fact of nature or a rule)? → Zero. Is it about something that might happen in the future if a condition is met? → First.

Memory Hook

Zero = 0 surprises. It always happens. It's a FACT.
First = the FIRST possible future. It WILL happen IF the condition is met.

Section 4

✍️ Worked Examples

Identify: "If plants don't get enough sunlight, they wilt."
1
Look at the IF clause: "plants don't get enough sunlight" → present simple ✓
2
Look at the result clause: "they wilt" → present simple (no will).
3
Meaning check: Is this always true? Yes — it's a fact of nature. → Zero Conditional.
✓ Zero Conditional — always-true fact of nature. Formula: If + present simple, present simple.
Identify: "If it rains tomorrow, we will cancel the picnic."
1
Look at the IF clause: "it rains tomorrow" → present simple ✓
2
Look at the result clause: "we will cancel the picnic" → will + base verb. Key word: will.
3
Meaning check: Is this a real future possibility? Yes — it might rain tomorrow. → First Conditional.
✓ First Conditional — real future possibility. Formula: If + present simple, will + base verb.
Fill the blank: "If she __________ the final round, she will win the championship." (pass)
1
Check the result clause: "she will win" → will + base. This is a First Conditional.
2
First Conditional rule: IF clause uses present simple.
3
Present simple of "pass" for "she" → passes.
✓ Answer: passes — "If she passes the final round, she will win the championship."

Summary

✅ What to Remember

Zero Conditional

  • Always-true facts and habits
  • IF + present simple
  • Result = present simple (no will)
  • Example: If you heat ice, it melts.

First Conditional

  • Real future possibility
  • IF + present simple
  • Result = will + base verb
  • Example: If it rains, we will stay inside.
Never forget

NEVER put will in the IF clause. The IF clause always uses present simple for both Zero and First Conditionals.
