👏 You understand the worlds — now we train the hunt

Last time you proved you GET it: when asked "which sentence is correct?" you found it and even fixed your own answer. The only thing left is the hunt — when a question says "which one has the ERROR?", you need to check all four, not stop at the first odd-looking one.

Before We Start 🔥

Warm-up 1 — which world?

"…had a longer runway…" — Maybe World or Too-Late World?

Warm-up 2 — the "will" rule

Spot the slip: "If the road had been salted, the cars will have stopped safely."

Warm-up 3 — the hunt habit

When a question asks "which sentence has the error?", the safest first move is to:

The hunt: label ALL four 🧦🧦🧦🧦

🟦 Maybe World

had + a thing → would + verb

"had a bigger net" → "would catch"

🟥 Too-Late World

had + a done-word → would have + verb

"had been mended" → "would have held"

You already know this. The error question is only hard because there are four sentences and the odd-socks one hides among them. The fix is a habit:

Label both halves of EVERY option first.
Then the one with two different worlds is the error.
Don't stop early
A "had + a thing" sentence looks odd but is usually fine (🟦🟦). If you grab the first odd-looking one you'll pick a correct sentence by mistake. Label all four, then decide.
Worked example — all four labelled

A. "had a wider gate" 🟦 + "would open" 🟦 = 🟦🟦 ✓
B. "had been locked" 🟥 + "would have stayed shut" 🟥 = 🟥🟥 ✓
C. "had a taller fence" 🟦 + "would have kept the deer out" 🟥 = 🟦🟥 odd socks → the error
D. "had been painted" 🟥 + "would have lasted" 🟥 = 🟥🟥 ✓

Drill 1 — Pair check (warm the hunt) 🧦

For each sentence, what do the two worlds make?

1. "If the shop had a side door, deliveries would arrive more quietly."
2. "If the boat had been bailed out, it would have stayed afloat."
3. "If the orchard had taller fences, the deer would have stayed out last year."
4. "If the choir had warmed up, the first song would have sounded clearer."

Drill 2 — The hunt (test format) 🎯

Find the odd-socks sentence (or, last set, the correct one). Label all four before you choose.

Set 1. Which sentence contains a grammar error?
Set 2. Which sentence contains a grammar error?
Set 3. Which sentence is grammatically CORRECT?
Label all four. The odd socks are the error.
Shortcut (optional): 🟦 Maybe = "2" (Second), 🟥 Too-Late = "3" (Third). Jot a tiny 2 or 3 above each half if it helps.
