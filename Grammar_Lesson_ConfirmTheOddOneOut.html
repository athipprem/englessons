Why this matters 🌍

Imagine a detective who picks a suspect just because they're dressed differently from everyone else at the scene — not because of any actual evidence. That's an easy mistake to make, but it's not real detective work. Real detective work means checking the EVIDENCE for every suspect before deciding. Spotting grammar errors works exactly the same way: a sentence that looks a bit different from its neighbours isn't automatically the "wrong" one. Today's lesson fixes exactly that one habit.

What actually happened at Unit 18 🔍

Three sentences all read "had been ___" (the Third recipe). One sentence read "had more sunlight" — no "been" at all. It stood out visually, so it got circled as the error. But run the three checks on it: no "will", no broken verb, and "had more sunlight" (past simple) matches "would ripen" (would+verb) perfectly — a completely clean Second Conditional. It just uses a DIFFERENT recipe, not a WRONG one. The real error was hiding in one of the "had been ___" sentences instead.

Watch it worked out — the confirm step in action 👀

"Which sentence contains a grammar error?" A. If the greenhouse had been heated, the seedlings would have survived the frost. B. If the pond had frozen overnight, the ducks would leave for warmer water. C. If the meadow had more wildflowers, the bees would visit more often. D. If the fence had been repaired, the sheep would have stayed safe.

First glance: A, B, and D all say "had been ___". C says "had more wildflowers" — no "been". It's tempting to circle C right away because it LOOKS different. Stop — that's not one of the three checks. Confirm it properly first.

Check C (the tempting one): "will"? No. Broken verb? No. Halves match? "had more wildflowers" (past simple) + "would visit" (would+verb) → matching Second. Passes all three — C is clean. It's just a different, equally valid recipe.

Now check the rest: A = Third-Third, clean. D = Third-Third, clean. B = "had frozen overnight" (had+past participle, Third if-half) + "would leave" (plain would, Second result) → halves DON'T match. B fails check 3 — that's the real error.

The lesson: looking different earned C a second look — that's fine. But only the three checks decide, and C passed all three. B never got the same length of noticing, but it was the one that actually failed. Confirm every candidate the same way, whether it "looks" suspicious or not.
