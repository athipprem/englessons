Mixed Conditional: Present Result

You know the Third Conditional. Now learn the one tiny change that makes it a Mixed Conditional — and why it matters.
🎯 ISA Priority — Grammar
Section 1 of 5

🔄 Quick Review: The Third Conditional

You already know this one well. Here is a quick reminder before we add the new layer.

The Third Conditional talks about an imaginary past — something that did NOT happen — and what the past result would have been.

Third Conditional Formula

If + had + past participle,    would have + past participle

Example: "If the team had trained harder, they would have won the championship." (They didn't train harder → they didn't win. Both past.)

Key idea for Third Conditional

Both the condition AND the result are in the past. The event is over. It is a "what if we could go back in time?" question.

Section 2 of 5

💡 What Changes in a Mixed Conditional?

The Mixed Conditional has the same if-clause as the Third Conditional — a past event that did NOT happen. But the result clause changes to show a present consequence.

In other words: the imaginary past event would still be affecting right now.

Type If-Clause (Condition) Result Clause Time of Result
Third If + had + past participle
If she had studied harder…		 would have + past participle
…she would have passed.		 PAST result
Mixed If + had + past participle
If she had studied harder…		 would + base verb (no "have")
…she would be a doctor now.		 PRESENT result
The One Change That Matters

Third Conditional result: would HAVE + past participle → "she would have passed" (past result)
Mixed Conditional result: would + BASE VERB → "she would be a doctor now" (present result)
Remove the word "have" from the result clause and the verb returns to base form.

Section 3 of 5

🔎 How to Spot a Mixed Conditional

The easiest way to recognise a Mixed Conditional is to look for time signal words that point to the present. These words tell you the result is happening now, not in the past.

now
direct present signal
today
refers to this day
still
ongoing to present
currently
happening at the moment

When you see any of these words in the result clause, that is your signal to use would + BASE VERB (not "would have + past participle").

The if-clause uses had + past participle (same as Third Conditional). The result uses would + base verb because "today" tells us the result is NOW.
The Quick Test

Does the result clause contain "now", "today", "still", or "currently"? → MIXED (use would + base verb). No time signal? → probably THIRD (use would have + past participle).

Section 4 of 5

📝 Worked Examples — Third vs Mixed

Worked Example 1 — Third Conditional (past result)
"If the captain _______ the warning signal, the ship _______ the rocks."
1
Is there a time signal like "now" or "today"? NO.
2
The result (missing the rocks) would have happened in the past — at the same time as the signal. Both past.
3
→ Third Conditional: had seen / would have avoided
✓ Answer: "If the captain had seen the warning signal, the ship would have avoided the rocks."
Worked Example 2 — Mixed Conditional (present result)
"If the ancient library _______ in the fire, scholars _______ access to thousands of lost texts today."
1
Time signal check: "today" → this is a PRESENT result.
2
The library burning happened in the past (if-clause = Third Conditional structure). But scholars having access is happening NOW.
3
→ Mixed Conditional: if-clause = had survived / result = would have — WAIT. "Today" is present. So result = would have access (NOT "would have had access").
✓ Answer: "If the ancient library had survived the fire, scholars would have access to thousands of lost texts today." — Note: "would have access" is would + base verb. "Would have" here is NOT "would have + past participle" — the base verb is simply "have" (= possess). This is the Mixed Conditional.
Worked Example 3 — spot the difference
Sentence A: "If she had taken the job, she would have earned more money."
Sentence B: "If she had taken the job, she would be living in London now."
1
Sentence A: no present time signal. Result is earning money in the past. → Third Conditional. ✓
2
Sentence B: "now" = present time signal. Living in London is a present state. → Mixed Conditional. ✓
3
Both have the same if-clause structure. The result clause is what changes.
✓ Key: Third = "would have earned" (past result) · Mixed = "would be living" (present result, with "now")

You can now distinguish Third from Mixed Conditional using the time signal test. Ready for Unit 5!

Third Conditional

  • If + had + past participle
  • would HAVE + past participle
  • Both condition and result in the PAST
  • No "now/today/still" in result

Mixed Conditional

  • If + had + past participle (same as Third)
  • would + BASE VERB (no "have")
  • Result is a PRESENT state or situation
  • Look for: now / today / still / currently

The Quick Test — memorise this

  • Does the result clause have "now", "today", "still", or "currently"? → MIXED → use would + base verb
  • No present time signal? → THIRD → use would have + past participle
  • The if-clause structure is identical for both types.
