Clause-Joiner Functions

You know what sentence-starters do. Now master the same three functions inside a single sentence: RESULT vs CONTRAST vs TIME at the clause-joiner level.
🎯 ISA Priority — Cloze Test
Section 1 of 5

🔎 What Is a Clause-Joiner?

A clause-joiner connects two parts of the same sentence. There is no full stop before the gap — the sentence continues.

You already know sentence-starters: words like However, Furthermore, Consequently — they come after a full stop and start a new sentence. Clause-joiners do the same job but inside one sentence, joining two clauses together.

The Period Test — Still First

Full stop before the gap? → Sentence-starter (However, Furthermore, Consequently...).
No full stop? → Clause-joiner (although, because, while, even though...).

In Unit 7, errors Q1 and Q4 were both clause-joiner slots. The structural type was correct — no full stop identified correctly — but the function was swapped between RESULT and CONTRAST. That is exactly what this lesson fixes.

Section 2 of 5

📚 The Three Clause-Joiner Families

Clause-joiners belong to the same three function families as sentence-starters. Here are the words you need to know for ISA:

🎯 RESULT clause-joiners
because, since, so (that), as
Test: "B happened BECAUSE OF A?" → YES
❓ CONTRAST clause-joiners
although, even though, though, while (concessive), whereas, despite
Test: "Is B surprising given A?" → YES
🕐 TIME clause-joiners
while (simultaneous), when, as, before, after, until
Test: "Are A and B happening at the same time?" → YES
Same Tests, New Setting

Because Backwards → YES = RESULT. Surprise Test → YES = CONTRAST. Simultaneous → YES = TIME (while/when). These tests work identically whether the gap is a sentence-starter or a clause-joiner.

Section 4 of 5

⚠️ The "While" Dual-Meaning Trap

The word while is unique — it can be either a TIME clause-joiner or a CONTRAST clause-joiner depending on the sentence. This is a common ISA trap.

⚠️ "While" — Two Completely Different Meanings
"She was cooking while the rain fell outside."
Are both things happening at the same time? YES → TIME while
✓ while = at the same time as (TIME)
"While she loved the city, she always missed the countryside."
Are both things happening at the same time? NO — these are contrasting feelings. Surprise Test: YES → CONTRAST while
✓ while = although / in contrast (CONTRAST)
"He watched the match while eating his lunch."
Simultaneous? YES → TIME while
✓ while = at the same time (TIME)
How to Decide Which "While"

First ask: "Are both things happening at the same time?" If YES → TIME. If NO → try Surprise Test. If the second part is surprising or contrasting with the first → CONTRAST while.

Clause-Joiner Decision Order

  • Step 1: Because Backwards → YES = RESULT (because)
  • Step 2: Simultaneous? YES = TIME (while, when)
  • Step 3: Surprising? YES = CONTRAST (although, even though)
  • Always test in this order — never skip

The "While" Dual-Meaning Alert

  • While = TIME when both happen at the same moment
  • While = CONTRAST when it means although/in contrast
  • Ask: simultaneous? If NO → try contrast meaning
  • This is a common ISA trap — always check both
