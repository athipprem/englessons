Before We Start — quick warm-ups 🔥

Warm-up 1

In "If the park had a duck pond…", the word after "had" is "a duck pond" — is that a thing, or a done-word (a verb like fixed, left, broken)?

Warm-up 2 — the "will" rule

Find the slip: "If the shed had been locked, nobody will have taken the tools."

Warm-up 3 — the ones that got away (Unit 11)

You were asked to find the error and picked a sentence that was actually fine. The real error mixed "had + a thing" with "would have + verb". Why is that a mistake?

That "different times" idea is the whole secret. Let's turn it into two easy pictures. Next →

Two worlds 🌍

Every "if" sentence lives in one of two worlds. Picture them:

🟦 Maybe World

Could still happen — like a daydream about now.

If-half: had + a thing ("had a bigger garden")

Result-half: would + verb ("would grow")

🟥 Too-Late World

Already over — can't change it now.

If-half: had + a done-word ("had been fixed", "had left")

Result-half: would have + verb ("would have grown")

🧦🧦 Both halves must be from the SAME world.
One of each is the mistake — like one short sock and one long sock.
See the slip

"If the orchard had taller trees, the birds would have nested higher."
"had taller trees" = a thing → 🟦 Maybe
"would have nested" → 🟥 Too-Late
🟦 + 🟥 = odd socks → this is the error. (To fix it: "would nest" keeps both in Maybe World.)

The trap to remember
"had + a thing" looks unusual, but it is not wrong by itself — it's just the Maybe World. Only blame a sentence when the two worlds clash.

Drill 1 — Which world? (one half at a time) 🌍

Just one little piece each time. 🟦 Maybe = a thing, or "would + verb". 🟥 Too-Late = a done-word, or "would have".

1. "…had a sunnier garden…"
2. "…had been repaired…"
3. "…would win more often."
4. "…would have melted."
5. "…had a louder voice…"
6. "…had left at dawn…"
7. "…would feel proud."
8. "…would have noticed."

Drill 2 — Do the two worlds match? 🧦

Check both halves. Same world = fine. Odd socks = the error.

1. "If the diner had a sunnier window, more people would stop to look."
2. "If the gate had been oiled, it would have opened quietly."
3. "If the dog had a longer lead, it would have reached the gate."
4. "If the cyclist had set off earlier, she would have finished before the rain."
5. "If the lounge had cosier chairs, the guests would stay chatting longer." (careful — "had + a thing" alarm!)
6. "If the orchestra had a bigger brass section, they would have filled the hall."

Drill 3 — Test format 🎯

Find the odd-socks sentence (or, in the last set, the correct one). Check both halves of every option.

Set 1. Which sentence contains a grammar error?
Set 2. Which sentence contains a grammar error?
Set 3. Which sentence is grammatically CORRECT?

Grown-up shortcut (only if you want it): teachers call 🟦 Maybe World "the Second Conditional (2)" and 🟥 Too-Late World "the Third Conditional (3)". You can jot a tiny 2 or 3 above each half — but the two pictures do exactly the same job.
