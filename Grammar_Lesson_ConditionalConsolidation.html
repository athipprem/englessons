Section 1 of 5

🚫 The Will Ban — Re-Hardening

In Unit 6, the Will Ban appeared secure. In Unit 7, it appeared in two wrong answers (Q11: will still have; Q13: will still be). The concept was not as solid as it looked. This section re-hardens it.

⛔ THE WILL BAN — ABSOLUTE RULE
"WILL" NEVER appears in Second, Third, or Mixed Conditional result clauses. Replace with "WOULD".
✗ If she had studied harder, she will pass the exam.
✓ If she had studied harder, she would pass the exam.
✗ If the bridge had not been built, the town will still be isolated today.
✓ If the bridge had not been built, the town would still be isolated today.
✗ If he had more free time, he will learn photography.
✓ If he had more free time, he would learn photography.
💡 Build the habit: when you see a conditional result clause in a test, scan for the word will before you read the meaning. If you see it — cross it out and eliminate that option immediately. Do not even read the rest of the option.
Why It Keeps Slipping

The Will Ban breaks down under pressure because "will" feels natural and confident in English. In everyday conversation, people say "if I win, I will..." all the time. The rule only applies to Second, Third, and Mixed conditionals — not First Conditional, where "will" is correct. Practice scanning for "will" before you read the meaning of each option.

First Conditional Exception

First Conditional (real future possibility): "If it rains, we will cancel the match." — "will" IS correct here. The ban only applies to Second, Third, and Mixed. Identify the if-clause type first, then check for "will".

Section 2 of 5

🔍 The Had Test — Second vs Third/Mixed

In Unit 6 Q8 and Unit 7 Q8, the same confusion appeared: classifying a Second Conditional as Third. The cause each time was the same — the if-clause contained "had" but it was not the past perfect.

Second Conditional
had + ADJECTIVE or NOUN
  • "had a larger hall" — larger = adjective ✓
  • "had more time" — more = adjective ✓
  • "had stronger leadership" — adjective ✓
  • "had a bigger budget" — noun phrase ✓
  • "had enough funding" — enough = adjective ✓
Third / Mixed Conditional
had + PAST PARTICIPLE VERB
  • "had studied" — studied = past participle ✓
  • "had not been closed" — been = past participle ✓
  • "had arrived earlier" — arrived = past participle ✓
  • "had not noticed" — noticed = past participle ✓
  • "had won the match" — won = past participle ✓
The Had Test — One Question

After "had," ask: "Is the next word a past participle verb?"
YES (had + studied / had + been closed / had + arrived) → past perfect → Third or Mixed Conditional.
NO (had + adjective / noun) → simple past of "have" → Second Conditional.

Quick Past Participle Check

Not sure if a word is a past participle? Try: "I have ___________." If it sounds correct, it is a past participle. "I have studied" ✓ → studied is past participle. "I have larger" ✗ → larger is an adjective, not past participle. "I have more" ✗ → more is an adjective.

Section 3 of 5

⚠️ Second + Third Mismatch — Error Spotting

In Unit 7 Q12, the task was: "Which sentence has a grammar error?" The error was a Second if-clause paired with a Third result — a mismatch. Prem chose a correctly formed sentence (C) instead of the error sentence (D).

This type of question requires you to apply the Had Test to every if-clause, then check whether the result clause matches:

If-clauseIf-clause typeResult clauseMatch?
"had not studied" (past participle ✓) Third/Mixed "would not have passed" (would have + past participle) MATCH ✓ Third
"had more time" (adjective — no past participle) Second "would write" (would + base verb) MATCH ✓ Second
"had more time" (adjective — Second) Second "would have written" (would have + past participle = Third result) MISMATCH ✗ Error!
"had not been closed" (past perfect — Third/Mixed) Mixed "would still be open today" (would + base + today) MATCH ✓ Mixed
"had more creative control" (adjective — Second) Second "would have produced" (Third result) MISMATCH ✗ Unit 7 Q12 error!
Error-Spotting Method — Two Steps

Step 1: Apply the Had Test to the if-clause of each option. Label it: Second, Third, or Mixed.
Step 2: Check the result clause. Does the result form match the if-clause label? If NOT → that is the error sentence.

The Trap in Error-Spotting Questions

Wrong answer options often look correct at a glance. Do not read for meaning — read for form. Apply the Had Test mechanically to every option, even the ones that sound correct. The error is usually in the mismatch between if-clause and result clause, not in meaning.

📖 Quick Reference — All Four Conditional Types

TypeIf-clauseResult clauseWhen to use
SecondIf + simple past (had + adj/noun)would + base verbImaginary present situation
ThirdIf + past perfect (had + past participle)would have + past participleImaginary past event, past result
MixedIf + past perfect (had + past participle)would + base verb + now/today/stillPast event, present-day result
FirstIf + present simplewill + base verbReal future possibility

Key rule: Mixed = past perfect if-clause + present time signal in result. Time signals = now, today, still, currently, at the moment.

🎉 All 5 Complete!

Three consolidation targets locked in: Will Ban, Had Test, Mismatch Error-Spotting.

The Three Fixes

  • Will Ban: scan for "will" — eliminate immediately in 2nd/3rd/Mixed
  • Had Test: next word a past participle? YES = Third/Mixed. NO = Second
  • Mismatch: Second if + Third result = ERROR (always)
  • First Conditional exception: "will" IS allowed with present-simple if

Mixed Conditional Signal

  • Past perfect if-clause (had + past participle)
  • Result clause = would + base verb
  • Must have: now / today / still / currently
  • No time signal? → probably Third, not Mixed
