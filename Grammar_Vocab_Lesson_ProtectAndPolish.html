Before We Start — quick confidence check 🎯

One quick error-scan warm-up with your pencil, exactly like last time.

1Which sentence contains an error?
1. If the ferry had left on time, the passengers would have made their connection.
2. If the storm had passed by morning, the beach would reopen for swimmers.
3. If the lighthouse had been repainted, it would have looked bright white again.
4. If the tide charts had been checked, the captain would have planned a safer route.
✏️ Keep using your pencil for every error-scan question from here on — practice AND the real test.

Quick reminder — the routine that worked 🔧

No new teaching today — just a reminder of exactly what worked last time, so it stays sharp.

For every option: write "3rd" above the IF-half if it says "had ___ed"; write "2nd" if it's a plain past. Write "3rd" above the RESULT-half if it says "would have ___ed"; write "2nd" if it's a plain "would ___". If the two labels for one option don't match, that's the error.
